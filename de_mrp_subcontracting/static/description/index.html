MRP Subcontracting

This module add feature is to provide MRP Subcontracting functionality enables to manage manufacturing services which are outsourced.

Work Center Configuration

Subcontracting Configuration will define in the Work Center(s) to activate outsourcing process with third party
Activate Subcontracting Process by activating "Is Subcontracting" flag active
Outsourcing Partner: The subcontracting service provider Partner.
Service Type Product: The Service Type Product use for Purchase Order and Vendor Invoicing.
Cost Per Unit: Cost Shall be calculated for Total Number of Unit of outsourced Product.
Picking Type: The outbound delivery type which use to send material to Partner for processing.
Return Type: The inbound delivery type which use to receive material from partner after processing.

Generate Subcontracting Order for Work Order

Auto Generate Subcontracting Document

It will create three document for subcontracting process.

Generate Purchase Order for Subcontracting Services

Subcontracting Purchase Order

Generate Outbound Document for Material Issues to Partner for Outsourced Processing

Generate Inbound Document to Receive Material after Processing from Outsourcing Partner
